Responsive Images:

  1. Density Switching
  2. Art Direction
  3. Resolution Switching

1- Density Switching:

This method is using HTML5 attribute srcset for <img srcset=""> tag to define different image sizes for different DPI (Device Pixel Ratio).


Example:

<img srcset="imgs/img-small1.jpg 1x, imgs/img-large1.jpg 2x">

1x descriptor is defined to load the small image for lower screen resolution devices. likewise 2x descriptor will load large image for larger screen resolution.

2- Art Direction:

This method is used to load different image sizes based on Media Queries defined for Browser Width rather than Screen Density.

In order to do Art Direction we need to use HTML5 <picture></picture> element where we need to specify 2 other elements <source> and <img> elements within the picture element as options.

Example: 
		
		<picture>
			// Browser Width Base Images 
			<source 
				srcset="imgs/img-500_2.jpg" 
				media="(max-width: 500px)">
			<source 
				srcset="imgs/img-700_2.jpg" 
				media="(max-width: 700px)">
			
			//Fallback Image with density resolution
			<img srcset="imgs/img-1440-1x_2.jpg 1x, imgs/img-1440-2x_2.jpg 2x">
		</picture>

3- Resolution Switching

In this method we force browser to choose the best image size based on Viewport and DPI.

This method just like density resoltion method we use srcset attribute for <img> tag where we also define the viewport width by using vw descriptor and size attribute to tell browswer which image size is the best for a viewport.
